## 📖 Project Overview

The **Python Smart Employee ID Generator** is a console-based application that automatically generates unique employee IDs using employee details such as name, department, and joining year.

This project demonstrates core Python concepts including variables, data types, operators, conditional statements, loops, strings, input/output operations, and validation techniques.

---

## ✨ Features

✅ Generate Unique Employee IDs

✅ Accept Employee Details from Users

✅ Validate Department Names

✅ Validate Joining Year

✅ Display Formatted Employee Information

✅ Support Multiple Employee Entries

✅ User-Friendly Console Interface
