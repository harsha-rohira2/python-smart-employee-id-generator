# Python Smart Employee ID Generator

## Project Overview

The Smart Employee ID Generator is a Python console-based application that automatically generates unique employee IDs using employee details such as name, department, and joining year.

This project demonstrates Python fundamentals including variables, data types, operators, conditional statements, loops, strings, input/output operations, and validation techniques.

## Features

- Generate unique Employee IDs
- Accept employee details from users
- Validate department names
- Validate joining year
- Display formatted employee information
- Support multiple employee entries
- User-friendly console interface

## Technologies Used

- Python 3
- VS Code

## Concepts Covered

- Variables and Data Types
- Input and Output
- Operators
- Conditional Statements
- Loops
- Strings and String Methods
- Membership Operators
- f-Strings

Employee Name: Harsha
